## 概要
railsチュートリアル(https://railstutorial.jp/) をベースに、なるべく最新の環境を用いて簡易SNSアプリを作成しました。<br>

## やったこと
Dockerを用いた環境構築<br>
railsを用いたアプリ作成<br>
AWSへのデプロイ<br>

## バージョン
Docker 20.10.17,/docker-compose 2.6.1<br>
Ruby 3.1.2<br>
Rails 7.0<br>
Bootstrap 5.1<br>
MySQL 8.0<br>

## 使用ツール
Visual Studio Code<br>

## 機能
ユーザー登録・表示・更新・削除<br>
ログイン・ログアウト<br>
メッセージの投稿<br>
ユーザーのフォロー<br>
